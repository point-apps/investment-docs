# Owners

## The owner model

The owner model contains all the information about your owners.

### Properties

<Properties>
  <Property name="_id" type="objectId">
    Unique identifier for the owner.
  </Property>
  <Property name="name" type="string">
    The name for the owner.
    - Unique Constraint
    - Is Required
  </Property>
  <Property name="createdBy_id" type="objectId">
    User identifier who create the owner.
  </Property>
  <Property name="createdAt" type="timestamp">
    Timestamp of when the owner was created.
  </Property>
  <Property name="updatedBy_id" type="objectId">
    User identifier who update the owner.
  </Property>
  <Property name="updatedAt" type="timestamp">
    Timestamp of when the owner was updated.
  </Property>
  <Property name="archivedBy_id" type="objectId">
    User identifier who archive the owner.
  </Property>
  <Property name="archivedAt" type="timestamp">
    Timestamp of when the owner was archived.
  </Property>
  <Property name="requestApprovalDeleteTo_id" type="objectId">
    User identifier who request to delete the owner.
  </Property>
  <Property name="requestApprovalDeleteAt" type="timestamp">
    Timestamp of when the user request to delete the owner.
  </Property>
  <Property name="requestApprovalDeleteReason" type="string">
    Request approval delete reason.
  </Property>
  <Property name="requestApprovalDeleteReasonReject" type="string">
    Reason to reject request delete.
  </Property>
  <Property name="requestApprovalDeleteStatus" type="enum">
    Request approval delete status
    - enum ['pending', 'approved', 'rejected']
  </Property>
</Properties>

## List all owners {{ tag: 'GET', label: '/v1/owners' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your owners.

    ### Optional query

    <Properties>
      <Property name="page" type="integer">
        The page number of owners returned.
        - Default `1`
      </Property>
      <Property name="pageSize" type="integer">
        The page size number of owners returned.
        - Default `10`
      </Property>
      <Property name="search" type="object">
        Filter data by text
      </Property>
      <Property name="sort" type="object">
        Display data in order.
        - `asc` sort ascending
        - `desc` sort descending
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/v1/owners">
    ```typescript {{ title: 'axios' }}
    const response = axios.get('/v1/owners', {
      params: {
        page:1,
        pageSize: 10,
        search: {
          name: 'john',
        },
        sort: {
          name: 'asc'
        }
      }
    })
    ```
    </CodeGroup>
    <CodeGroup title="Response">
    ```json {{ title: '200' }}
    {
      "owners": [
        {

        },
        {
          "_id": "637d83d15d2be122007524bf",
    ```
